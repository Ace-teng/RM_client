# HUD 布局位置规范

边缘化布局：中央区域仅显示图传画面与准心，所有 UI 贴边放置。

## 布局示意图

```
┌─────────────────────────────────────────────────────────────────────┐
│ [己方血量] [电容条] [自瞄状态]    [7:00]    [能量限制] [对方血量]   │
│ [发弹量]                                                             │
├─────────────────────────────────────────────────────────────────────┤
│ ┌─────────┐                                    ┌─────────┐          │
│ │ 左侧边栏 │                                    │ 右侧边栏 │          │
│ │ • 等级   │      【 图 传 画 面 】             │ • 哨兵   │          │
│ │ • 性能   │        [准心 + 飞坡线]             │ • 信息   │          │
│ │ • Buff  │        [自瞄框]                    │         │          │
│ └─────────┘                                    └─────────┘          │
├─────────────────────────────────────────────────────────────────────┤
│ ┌─────────┐ [底盘状态] [热量条]                    ┌─────────┐       │
│ │ 小地图  │ [跳跃目标] [技能/状态栏]               │ 悬浮窗  │       │
│ │ 160×160│ [大小符提醒]                           │ 信息面板│       │
│ └─────────┘                                       └─────────┘       │
└─────────────────────────────────────────────────────────────────────┘
```

## 位置与 CSS 类名对照

| 区域 | 元素 | CSS 类名 |
|------|------|----------|
| 顶部左 | 己方血量 | `.position-top-left` |
| 顶部左 | 电容条 | `.position-top-left-2` |
| 顶部左 | 自瞄状态 | `.position-top-left-3` |
| 顶部中 | 比赛时间 | `.position-top-center` |
| 顶部右 | 能量限制 | `.position-top-right-2` |
| 顶部右 | 对方血量 | `.position-top-right` |
| 顶部右 | 发弹量 | `.position-top-right-3` |
| 中央 | 准心 | `.position-center` |
| 左侧 | 左侧边栏（等级/性能/Buff） | `.position-sidebar-left` |
| 右侧 | 右侧边栏（哨兵/信息） | `.position-sidebar-right` |
| 底部左 | 小地图 160×160 | `.position-bottom-left` |
| 底部左 | 底盘状态 | `.position-bottom-left-2` |
| 底部左 | 跳跃目标 | `.position-bottom-left-3` |
| 底部左 | 大小符提醒 | `.position-bottom-left-4` |
| 底部中 | 技能栏 | `.position-bottom-center` |
| 底部右 | 热量条 | `.position-bottom-right-2` |
| 底部右 | 悬浮窗/信息面板 | `.position-bottom-right` |

## 层级（z-index）

在 `styles/hud.css` 中定义，使用类名或变量：

| 层级 | 类名 | z-index | 用途 |
|------|------|---------|------|
| 基础 | `.z-base` | 100 | 普通 HUD 元素 |
| 准心 | `.z-crosshair` | 150 | 准心、瞄准框 |
| 警告 | `.z-alert` | 200 | 警告弹窗 |
| 模态 | `.z-modal` | 300 | 模态窗口 |
